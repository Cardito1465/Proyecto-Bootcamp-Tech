Energía Solar : La única energía que prevalecerá millones de años sin importar lo que hagamos los humanos.

Objetivo general
Desarrollar una página web informativa sobre energía solar utilizando HTML, CSS y JavaScript, que permita divulgar contenido educativo, interactivo y visualmente atractivo, con el fin de promover el conocimiento sobre energías renovables y su aplicación en la vida cotidiana

Objetivos específicos:
•	Construir la arquitectura del blog mediante HTML, organizando secciones como inicio, información técnica, tipos de energía solar, beneficios, artículos y contacto.
•	Aplicar CSS para crear una interfaz atractiva, funcional y adaptable a dispositivos móviles, utilizando colores, tipografías y elementos gráficos relacionados con la energía solar.
•	Desarrollar componentes dinámicos como menús desplegables, animaciones, validación de formularios y visualización interactiva de contenido para mejorar la experiencia del usuario.
•	Crear y organizar textos, imágenes e infografías que expliquen qué es la energía solar, cómo funciona, sus beneficios, aplicaciones y tendencias tecnológicas actuales.
•	Asegurar que la estructura, enlaces, botones y secciones sean fáciles de usar, garantizando que cualquier usuario pueda explorar la información de manera clara y fluida.
